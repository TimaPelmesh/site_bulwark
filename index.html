Мониторинг системы и сети в одном приложении

Приложение для мониторинга системных ресурсов компьютера и сетевой активности с графическим интерфейсом на PyQt5

Скачать приложение

Функциональность

Системный мониторинг

  • Информация о системе (ОС, процессор, ядра)
  • Мониторинг загрузки процессора
  • Мониторинг использования оперативной памяти

Сетевой мониторинг

  • Информация о сетевых интерфейсах
  • Информация о сетевом шлюзе
  • Статистика по сетевому трафику
  • ARP таблица (кэш адресов)
  • Визуализация топологии сети

Структура проекта

.
├── main.py              # Основной файл для запуска приложения
├── ui.py                # Модуль с основным пользовательским интерфейсом
├── system_monitor.py    # Модуль для мониторинга системных ресурсов
├── network_monitor.py   # Модуль для мониторинга сетевой активности
├── icon.ico             # Иконка приложения
└── README.md            # Этот файл с документацией

Требования

Для запуска приложения необходимы следующие компоненты:

  • Python 3.6 или выше
  • PyQt5
  • psutil
  • netifaces
  • scapy
  • networkx
  • matplotlib
  • GPUtil (для мониторинга GPU)

Установка и запуск

1. Установка зависимостей

pip install PyQt5 psutil netifaces scapy networkx matplotlib GPUtil

2. Запуск приложения

python main.py

Использование

Системный мониторинг

  1. Вкладка "Система" - отображает основную информацию о системе.
  2. Вкладка "Процессор" - показывает загрузку ЦП в реальном времени.
  3. Вкладка "Память" - отображает информацию о RAM и дисках.

Сетевой мониторинг

  1. Вкладка "Сетевые интерфейсы" - показывает информацию о сетевых адаптерах, их настройках и статистике трафика.
  2. Вкладка "ARP таблица" - отображает таблицу ARP-записей (IP-адреса и соответствующие им MAC-адреса).
  3. Вкладка "Топология сети" - визуально отображает топологию локальной сети с обнаруженными устройствами.

Принципы ООП в проекте

Инкапсуляция

Данные и методы объединены в классы, скрывая детали реализации.

Наследование

Классы виджетов наследуются от базовых классов PyQt (QWidget, QTabWidget).

Полиморфизм

Одни и те же методы (например, closeEvent) имеют разную реализацию в разных классах.

Абстракция

Классы представляют абстрактные концепции, такие как "Сетевой мониторинг" или "Системный мониторинг".

Примечания

Приложение разработано для работы в операционной системе Windows.

Для некоторых функций (например, сканирование сети) могут потребоваться права администратора.
